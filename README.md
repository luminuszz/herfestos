## Herfestos CLI

### Simple cli for create componentes with styled-components

![alt](./assets/hefesto.jpg)

#### Config

```json
{
  // herfestos.config.json
  "componentsDir": "./src/components"
}
```

#### install

```bash
 npm i -g ultron-cli
 or
 yarn global add ultron-cli

```

#### Use

- bash: `her or herfestos create Button`
